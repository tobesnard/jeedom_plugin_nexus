# Installation des icônes Edom

1) Création d'un pack d'icône sur [icoMoon](https://icomoon.io/new-app/nexus) 
    ou chargement du projet nexus
2) Suivant la version d'icomoon 
    - (Ancienne version) `Générer de la font` en sélectionnant les options :
        - Font Name : nexus
        - Préfixe : nexus-
        - CSS Selector : Use a attribute selector
    - (Nouvelle version d'icomoon) :
        - selectionné formats>font
        - Font > Font Family : `nexus`
        - Font > CSS Settings > `Define a Class per Glyph`
        - Font > CSS Settings > Prefix : `nexus-`
3) Renommer le fichier zip  `nexus.zip`
4) Optionnel, ne conserver que le fichier `style.css` et de répertoire `font`
